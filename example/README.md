# Examples

This directory contains small, publishable examples that can be used in the
project README, docs, and release notes.

## Comparison Result 4.33%

Open `comparison-4.33.html` in a browser to see a side-by-side render
comparison:

- Original SVG render
- Generated HTML render
- Optional pixel diff image

The image assets live in `example/assets/` so the example is self-contained and
does not depend on local `workspace/` artifacts.

`comparison-4.33.png` is a static screenshot of the same example for README
embedding.

The original SVG is `Group 2147208102.svg`. It comes from the default project
template on Tencent Ardot: <https://ardot.tencent.com/>.
